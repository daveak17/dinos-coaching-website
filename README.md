# Performance Coaching Website

## Overview
Conversion focused performance coaching website built in WordPress and deployed on a self managed VPS.

The goal of the site is lead generation through a structured landing page and a coaching application form.

## What I delivered
- Modern hero section with clear positioning and call to action
- Audience fit section showing who the program is for and not for
- Program breakdown with three pillars and a 12 week journey section
- Coach credibility section
- Application form with reCAPTCHA for spam protection
- Layout polish, spacing improvements, and responsive behavior for mobile

## Tech stack
- WordPress
- Kadence Theme and Kadence Blocks
- Ubuntu VPS
- Nginx
- PHP FPM
- MariaDB
- reCAPTCHA

## Deployment highlights
- DNS configured to route a custom subdomain to the VPS
- Nginx virtual host configured for WordPress
- File permissions set for a stable production setup
- Basic server validation with nginx testing and reload
